# How to use?
Put your obfuscated php file down in `PHP-Deobfuscator\bin\Debug\net6.0` this path(be careful the name must be `obfuscated.php`) and open `PHP-Deobfuscator.sln` with Visual Studio and just run the compiler directly. Then you'll find a deobfuscated file named `deobfuscated.php` at the same place with `obfuscated.php`.
